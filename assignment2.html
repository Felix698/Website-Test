Section 1.1 - What's Going Into my Webpage

Each of my webpages will have a solid background and white text in the Gill Sans font. The main page will have an ‘about me’ section, in addition to links leading to pages detailing the assignments. These sub-pages will have a top bar containing buttons, one for returning to the home page, and others for quickly navigating between different sections of the assignment. The sections will be organised in sequential order, and every few sections there will be a link that can be clicked to return to the top of the page.

Section 1.5 - Copyright and Fair Use

When collecting research and images for a site, one is almost guaranteed to run into copyrighted content. Put simply, this is content that is legally owned by an individual or company (the copyright holder) and to use it requires permission, either by a spoken agreement or exchange of royalties. If an individual fails to comply with this, they risk being liable to legal action by the copyright holder.
There are some exceptions, however, to this law. One such exception is Fair Use - if copyrighted content is used in a transformative and limited enough way, or is not used for monetary gain, the one using can be exempt from litigation. Content that falls under fair use can include works of parody and assets used in non-commercial projects with express credit given.

Section 1.6 - User Disabilities and Impairments

In general, most disabilities do not have too severe an impact on one’s ability to use the web. However, there is one very notable group of exceptions - sight-based disabilities.
For example, a colourblind user may have difficulty navigating pages that have certain or low contrasting colours, so a monochrome colour palette may be preferable. It is also possible that the user has partial or total blindness, which they can work around by having a program to read aloud text on the webpage. A developer can help make this program more effective by adding descriptions into the code of certain assets such as pictures, which it can relay to the user.
Another, much more dangerous affliction is photosensitive epilepsy, which can be worked around by using a consistent, low-saturation colour scheme and no bright flashes to prevent affected users suffering from a seizure.
